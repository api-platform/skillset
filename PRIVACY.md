# Privacy Policy

The API Platform Skillset plugin is a collection of local Markdown skill files for [Claude Code](https://code.claude.com).

## Data collection

This plugin does **not** collect, store, transmit, or process any personal data.

- No telemetry.
- No analytics.
- No network requests of its own.
- No user data is read, retained, or shared.

The skills are static documentation loaded locally by Claude Code. Any data handling is performed by Claude Code itself and is governed by [Anthropic's Privacy Policy](https://www.anthropic.com/legal/privacy).

Questions: open an issue at <https://github.com/api-platform/skillset/issues>.
